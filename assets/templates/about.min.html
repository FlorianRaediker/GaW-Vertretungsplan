Datenschutzerklärung

Allgemein

Diese Website ist datenschutzfreundlich konzipiert. Personenbezogene Daten werden nicht dauerhaft gespeichert, es findet kein ausgefeiltes Tracking statt, Nutzer*innen wird nicht hinterherspioniert, Drittdienste werden nicht genutzt und es wird keine Werbung geschaltet. Alle erfassten, länger gespeicherten Daten können transparent hier eingesehen werden.

Allgemeine Daten, die beim Aufrufen dieser Website erfasst werden

Folgende allgemeine Daten werden beim Aufrufen dieser Website automatisch von Ihrem Browser mitgesendet:

  • Ihre IP-Adresse
  • Datum und Uhrzeit des Aufrufs
  • Die aufgerufene Webseite („URL“) (evtl. ausgewählte Klassen bzw. Kürzel sind Teil der URL, werden aber nicht gespeichert, da es sich um personenbezogene Daten handelt)
  • Die Art des Webbrowsers, das verwendete Betriebssystem und ggf. verwendete Hardware („User-Agent“)
  • Die zuvor aufgerufene Webseite („Referrer“)

Ohne diese Daten, insbesondere die IP-Adresse, ist es technisch nicht möglich, auf die Website zuzugreifen.

Die Daten werden für 14 Tage in Server-Logfiles gespeichert und danach gelöscht. Die Speicherung erfolgt aufgrund meines berechtigten Interesses gemäß Art. 6 Abs. 1 lit. f DSGVO an der Sicherheit und Verfügbarkeit dieser Website.

Weitere Daten, die beim Aufrufen dieser Website erfasst werden

Folgende weitere Daten werden beim Aufrufen dieser Website zu Analysezwecken für längere Zeit gespeichert. Die Daten sind nicht personenbezogen.

  • Datum und Stunde des Aufrufs
  • Die aufgerufene Webseite („URL“)
  • Die Art des Webbrowsers, das verwendete Betriebssystem und ggf. verwendete Hardware („User-Agent“)
  • Die zuvor aufgerufene Webseite („Referrer“)
  • Ob Vertretungen mittels Klassen bzw. Kürzeln ausgewählt sind. Es wird nur erfasst, ob diese Funktion genutzt wird. In keinem Fall werden die ausgewählte Klasse bzw. das ausgewählte Kürzel länger gespeichert als nötig, um die angezeigte Webseite zu generieren.
  • Ob Benachrichtigungen eingeschaltet sind
  • Ob Stundenpläne eingetragen wurden. Im Gegensatz zu Klassen bzw. Kürzeln wird der Stundenplan ausschließlich in Ihrem Browser gespeichert und niemals an den Server gesendet.
  • Welches Design verwendet wird (hell oder dunkel)

Diese Daten werden erfasst, um die Nutzung von Funktionen zu analysieren und die Website entsprechend optimieren zu können.

Widerspruch

Das Erfassen oben genannter weiterer Daten wird durch aktuelle Ad-und-Tracking-Blocker wie uBlock Origin (Firefox, Chrome, Edge, Opera) blockiert. Die Website respektiert diese Nutzer*innenentscheidung und es wird nicht versucht, das Blockieren zu umgehen.

Transparenz

Alle erfassten weitere Daten können transparent hier eingesehen werden. Diese Daten sind nicht personenbezogen — personenbezogene Daten werden nie dauerhaft gespeichert.

Keine Tracking-Cookies

Diese Website kommt ohne Tracking-Cookies aus.
Es wird genau ein Cookie pro Vertretungsplan genutzt, um die ausgewählte Klasse bzw. das ausgewählte Kürzel zu speichern. Bei einem erneuten Aufruf kann so dieselbe Klasse bzw. dasselbe Kürzel automatisch ausgewählt werden.

TLS-Verschlüsselung

Alle Daten, die zwischen Ihrem Browser und dem Server dieser Website ausgetauscht werden, sind mit Verschlüsselungsverfahren vor Dritten geschützt. Die Verschlüsselungsverfahren (TLSv1.2 und TLSv1.3) entsprechen dem Stand der Technik. Es sei darauf hingewiesen, dass ein lückenloser Schutz nicht möglich ist.

Benachrichtigungen

Optional können pro Vertretungsplan Push-Benachrichtigungen eingeschaltet werden, die bei neuen Vertretungen verschickt werden. Es gibt keine Garantie, dass diese Benachrichtigungen zuverlässig verschickt werden.

Benachrichtigungen werden über einen Dienst Ihres Browsers versendet. Es gelten die entsprechenden Datenschutzbestimmungen des Browserherstellers. Der Inhalt der Benachrichtigungen wird verschlüsselt übertragen und kann daher vom Browserhersteller nicht eingesehen werden.

Wenn keine Klassen bzw. keine Kürzel ausgewählt sind, ist die Nutzung ohne Angabe personenbezogener Daten möglich. Vom Server dieser Website wird nur eine ID erfasst, die der Browserhersteller Ihrem Gerät zuordnen kann, um die korrekte Zustellung der Benachrichtigung zu ermöglichen.
Wenn Klassen bzw. Kürzel ausgewählt sind, wird diese Auswahl ebenfalls auf dem Server gespeichert, damit Sie nur Benachrichtigungen für ausgewählte Vertretungen erhalten.
Die Daten werden gelöscht, sobald Sie Benachrichtigungen auf der Webseite deaktivieren.

Änderung dieser Datenschutzerklärung

Diese Datenschutzerklärung kann von mir gegebenenfalls angepasst werden. Bei größeren Änderungen wird vorab auf der Website prominent darauf hingewiesen.
